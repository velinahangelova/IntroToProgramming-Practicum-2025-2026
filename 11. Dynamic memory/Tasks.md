### Динамична памет
--------------------------------------------------
Теоретични задачи:
---

1. Какво ще изведе следният код?

```cpp
#include <iostream>

int main() {
int *p = new int(7);
int *q = p;

*q = 9;
cout << *p << endl;
delete p;
}
```

2. Какво ще изведе следният код?

```cpp
#include <iostream>

int main() {
int *p = new int[2];
p[0] = 5;
p[1] = 10;

delete[] p;
cout << p[1] << endl;

}
```

3. Какво ще изведе следният код?

```cpp
#include <iostream>

int main() {
int *p = new int[3];
p[0] = 1;
p[1] = 2;
p[2] = 3;

cout << p[1] << endl;
}
```

Практически задачи:

**0.** Напишете програма за следенето на резултатите на отбори. Всеки играч има точки и всеки отбор може да има различен брой играчи.  
Имате възможност за добавяне на точки на играч (по индекс на отбор и играч). След всяка актуализация се отпечатват точките на играчите от всеки отбор, като отборите са сортирани по сумата на точките на играчите си.

**1.** Напишете функция, която приема масив от цели числа и елемент на масива. Функцията преподрежда елементите, така че всички по-малки елементи от подадения да са вляво, а всички по-големи — вдясно.

Вход:
```
4 2 7 5 3
5
```
Изход:
```
4 2 3 5 7
```

**2.** Напишете функция, която разбива изречение на думи.Дума е последователност от малки и главни латински букви.

Вход:
```
Programming123 is fun!
```
Изход:
```
Programming
is
fun
```

**3.** Напишете функция, която приема число и го конвертира в символен низ. Функцията връща указател към първия елемент на масива със символи.

Вход:
```
12345
```
Изход:
```
"12345"
```

**4.** Напишете функция, която приема два сортирани във възходящ ред символни низа и символен низ `result`. Функцията обработва `result` така, че да представлява **сортираното сливане на двата низа**.

Вход:
```
aadgghxyyz
abcfijkl
```
Изход:
```
aaabcdfgghijklxyyz

**5.** Напишете функция, която приема символен низ от цифри (максимум 256 символа) и символен низ `result` (максимум 256 символа). Функцията обработва `result` така, че да съдържа **информация за броя на всяка цифра**.  

Вход:
```
1122334455
```
Изход:
```
1:2 2:2 3:2 4:2 5:2
```

**6.** Напишете функция, която приема символен низ с дължина най-много 64 символа и премахва от него всички символи, които не са цифри, и след това превръща обработения низ в число.

Вход:
```
a1b2c3
```
Изход:
```
123
```

**7.** Подредица (subsequence) на дадена редица (sequence) наричаме редица, която се получава от дадената редица, чрез изтриване на елементи без да се променя наредбата на оставащите елементи. Например, редицата (A, B, E) е подредица на (A, B, C, D, E, F) след премахване на елементите C, D, F. Напишете функция, която приема два символни низа с дължина най-много 256 символа и връща дали първият е подредица на втория.

Вход:
```
abe
abcdef
```
Изход:
```
true
```

Вход:
```
bad
gabd
```

```
false
