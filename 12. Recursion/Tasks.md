### Рекурсия
--------------------------------------------------
Теоретични задачи:
---

1. Какво ще изведе следният код?

```cpp
#include <iostream>
using namespace std;

void f(int n) {
    if (n == 0) return;
    f(n - 1);
    cout << n << " ";
}

int main() {
    f(3);
}
```
2. Какво ще изведе следният код?

```cpp
#include <iostream>
using namespace std;

void g(int n) {
    if (n <= 0) return;
    cout << n << " ";
    g(n - 2);
    cout << n << " ";
}

int main() {
    g(5);
    return 0;
}
```

Практически задачи:
---

**1.** Да се реализира рекурсивна функция, която пресмята n!


**Пример:**


Вход:
```
5
```
Изход:
```
120
```

**2.** Да се реализира рекурсивна функция, която пресмята n-тото число на фибоначи.(редицата започва от 1)


**Пример:**


Вход:
```
4
```
Изход:
```
3
```

**3.** Да се реализира рекурсивна функция, която вдига число на дадена степен.


**Пример:**


Вход:
```
3 4
```
Изход:
```
81
```

**4.** Да се реализира рекурсивна функция, която приема масив от цели числа и връща сумата на елементите в него.


**Пример:**


Вход:
```
3 4 5 2 1 7 8
```
Изход:
```
30
```

**5.**  Да се реализира рекурсивна функция, която приема сортиран масив и число и връща дали числото се съдържа в масива. (Двоично търсене).


**Пример:**


Вход:
```
3 4 5 2 1 7 8
6
```
Изход:
```
false
```

Вход:
```
3 4 5 2 1 7 8
8
```
Изход:
```
true
```

**6.**  Да се реализира рекурсивна функция, която приема стринг и връща дали стрингът е палиндром.


**Пример:**


Вход:
```
eopoer
```
Изход:
```
false
```

Вход:
```
eopoe
```
Изход:
```
true
```

**7.**  Да се реализира рекурсивна функция, която приема стринг и връща броя на малките и големите символи в него.


**Пример:**


Вход:
```
eoPoEr
```
Изход:
```
lower - 4
upper - 2
```

**8.**  Да се реализира рекурсивна функция, която приема стринг и обръща малките бувки в главни и главните в малки.


**Пример:**


Вход:
```
eoPoEr
```
Изход:
```
EOpOeR
```
